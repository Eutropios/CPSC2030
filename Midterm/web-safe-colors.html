Web Safe Colors

Web safe colors consist of set of 216 that were designed to display consistently in all monitors and web browsers back in the era of 256-color (8-bit) displays.
Web safe colors are represented using hexadecimal code and are made of all the combinations of 00, 33, 66, 99, CC, and FF for RGB values (e.g., #0033CC)
