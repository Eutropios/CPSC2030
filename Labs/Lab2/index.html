MIME

Originally developed to support text in character sets other than ASCII, MIME now enables images, audio, video, and any binary data to be attached to emails.
MIME is a specification for formatting non-ASCII messages so that they can be sent over the Internet.

Beyond emails, MIME has become an integral part of the wbe and various internet protocols such as HTTP.

Base64

Base64 encoding is commonly used in the context of MIME to encode binary data so that it can be safely transmitted via text-based TCP/IP protocols such as SMTP and HTTP.
Base64 encoding converts binarybinary data into a set of ASCII characters. It divides the non-ASCII data into 6-bit chunks, each of which is then represented by one of the 64 possible ASCII characters.
